# Security Lab: Metasploit Framework Research

This repository is dedicated to my personal research on penetration testing and vulnerability analysis using the Metasploit Framework.

## Objective
* **Mechanism Analysis:** Understanding exploit mechanisms in localized, controlled environments.
* **Skill Enhancement:** Improving security posture through ethical hacking practices and deep-dive protocol analysis.
* **Documentation:** Managing custom configurations, auxiliary modules, and shellcode integration for future security audits.

## Environment & Tech Stack
* **Target Architecture:** x86/x64 (C-based vulnerability research)
* **OS:** Kali Linux on WSL2 (Windows Subsystem for Linux)
* **Framework:** Metasploit Framework (MSF)
